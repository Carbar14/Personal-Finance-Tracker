Importing Data from Excel or Google Sheets

If you have data in an Excel document or a Google Sheet that you would like to import into our system, you can easily do so by following these steps:

  1. Export to CSV:
    • Open your document in either Excel or Google Sheets.
    • Export the file as a CSV (Comma-Separated Values) file.
  2. Ensure Correct Column Names (case insensitive):
    • Before exporting, ensure that your data includes the same column names as shown in the examples below for incomes and expenses.

Example Column Names for Expenses:

  • Category
  • Description
  • Purchase Location
  • Quantity
  • Price
  • Date
  • Color

Example Column Names for Incomes:

  • Category
  • Description
  • Method
  • Income
  • Date
  • Color

Note: Columns can be blank except income, price, quantity, and date.

  1. Importing the CSV File:
    • Navigate to the import section of our platform.
    • Choose the CSV file you just exported.
    • Follow the prompts to complete the import process.
